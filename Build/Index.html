  • Calk

    Calk est un projet réalisé dans le cadre des éléctions pour la BDE de l'IUT informatique de Bordeaux visant à fournir un support d'édition et de partage de cours collaboratif.

    Son fonctionnement est basé sur l'idée de calques. Un calque correspondant à une version d'un cours. Un calque peut être 'forké' et par là amélioré.

  • IUT-Stajin

    Stajin est une application de gestion de Stage développée pour le compte de l'Institut Universitaire de Technologie de Bordeaux.

    Il se découpe en deux modules distincts (DataUtils et Client).

    DataUtils prend la forme d'un script python chargé de traiter les données et de gérer les éventuelles erreurs.
    Client consiste en une interface web chargée d'afficher différentes statistiques sur les données précedemment traitées.

  • Pokemon-Go WebAPI

    Ce projet vise à fournir une API JSON permettant de récupérer des informations depuis les serveurs du jeu Pokémon GO afin de les rendre accessible depuis n'importe quel système connecté à internet.

    Ce système est basé sur le travail fournit par 'Tejado', pgoapi.
